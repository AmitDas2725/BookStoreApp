# FindBook 📚

This is a full-stack bookstore application I'm building. It uses **React** for the frontend and **Node/Express** for the backend, with **MongoDB** as the database.

I've organized the project into two main folders: `FRONTEND` and `BACKEND` to keep the code clean and manageable.

---

### 🚀 Key Features
- **Frontend:** Built with React and Tailwind CSS.
- **Backend:** Node.js server handling the API and database logic.
- **Database:** MongoDB integration for storing book information.
- **Theme:** Includes a light/dark mode toggle.

---
